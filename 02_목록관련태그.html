ul(Unordered List)태그 : 순서가 없는 목록

li(List Item) 태그 : 목록의 한 줄

  • 김밥
  • 샌드위치
  • 햄버거
  • 제육볶음

ol(Ordered List) : 순서가 있는 목록

    - type 속성 : 표시되는 숫자 형식 지정

    type="a" : 영어 소문자(a, b, c, d...)
    type="A" : 영어 대문자(A, B, C, D...)
    type="I" : 로마자(I, II, III, IV...)
    type="i" : 로마자(i, ii, iii, iv...)

    start 속성 : 목록 시작번호

    reversed 속성 : 순서뒤집기
  1. HTML5
  2. CSS3
  3. JavaScript
  4. Java
  5. DB(ORACLE)

Emmet : HTML, CSS 빠른 자동완성을 지원하는 플러그인

    태그명 : 태그 자동 완성 ex) h1 pre p i strong 등

    > : 태그 내부에 작성 (하위)

    + : 태그 옆에 작성(형제)

    * 숫자 : 숫자 만큼 생성(곱하기) ex) *4는 4개생성

    {} 중괄호: 태그 내부에 작성될 내용

    $ : 1부터 1씩 증가하는 숫자
